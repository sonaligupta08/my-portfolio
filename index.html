CryptoConnect

CryptoConnect is a decentralized social media prototype where users can create posts and communicate through real-time chat. When a post is deleted, the content is removed from the UI while its cryptographic hash is preserved in storage to maintain a tamper-evident record.

  • 🔹 Real-time messaging using Socket.io
  • 🔹 Ethereum wallet authentication with MetaMask
  • 🔹 Local blockchain testing using Ganache
  • 🔹 Hash-based post integrity verification
